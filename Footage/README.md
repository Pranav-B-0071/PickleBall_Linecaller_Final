# Footage

Put the three camera videos here as:

- `cam1.mp4`
- `cam2.mp4`
- `cam3.mp4`

On page 3 of the web app, press the Calculate button to run tracking and
rendering. The generated files, such as `cam1_tracked.mp4` and
`cam1_tracked.csv`, are saved in this folder.

Those generated tracked videos and CSV files can then be used directly by the
Analyse button to show the bounce statistics.
